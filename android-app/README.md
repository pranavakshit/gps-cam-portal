# Android App (Kotlin)

This directory is reserved for the Kotlin-based Android application.

## Initialization Instructions

To start developing the Android app:
1. Open **Android Studio**.
2. Select **File > New > New Project**.
3. Choose **Empty Activity** (or another suitable template).
4. Set the project location to this `android-app` folder.
5. Set Language to **Kotlin** and Minimum SDK to an appropriate version (e.g., API 24).
6. Click **Finish**.

Android Studio will automatically generate the Gradle wrapper and project structure.
